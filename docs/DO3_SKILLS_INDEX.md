# DO3 Skills Index

Краткий индекс навыков для задач по 1С:Документооборот 3.0.

## Как пользоваться

- Сначала определите ожидаемый результат (что именно нужно получить).
- Выберите один основной навык из таблицы.
- Если задача смешанная, используйте `do3-scripts-router`.

## Маршрутизация задач

| Тип задачи | Какой навык использовать |
|---|---|
| Неясно, какой сценарий выбрать; нужно быстро классифицировать запрос | `do3-scripts-router` |
| Подстановка данных в шаблон файла, теги, визы, автозаполнение | `do3-file-autofill-scripts` |
| Скрипт шага схемы обработки объекта (документа), файлы по роли, запуск операций | `do3-object-processing-scripts` |
| Вычисление участников/ответственных в комплексном процессе | `do3-complex-process-scripts` |
| Поиск аналога в библиотеке скриптов Лушникова и адаптация паттерна | `do3-script-library-analyst` |
| Вставка регистрационного штампа и/или штрихкода в файл | `do3-stamp-and-barcode-scripts` |
| Бесшовная интеграция, пользовательские обработчики событий, интеграционные вызовы | `do3-integration-scripts` |

## Полезные замечания

- В скриптах схем и обработчиков учитывайте ограничения безопасного режима.
- Ошибки шага схемы для бизнес-процесса поднимайте через `ВызватьИсключение` с понятным текстом.
- Для запросов используйте параметры (`УстановитьПараметр`) и минимальный набор полей.
