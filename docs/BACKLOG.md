# Бэклог доработок расширения «ИИ анализ документов (Ollama)»

Список идей и незакрытых тем **без обязательства сроков**. Приоритеты уточняются вместе с заказчиком.

## Улучшение качества ответа и UX

- Расширить разбор Markdown → DOCX: вложенные списки, таблицы, гиперссылки, кодовые блоки (сейчас поддержано подмножество для юридических/структурированных ответов).
- Настраиваемый шаблон оформления DOCX (шрифты, цвета заголовков) через константы или регистр настроек.
- Явное разделение в UI: «очередь ИАДО» vs мониторинг штатного OCR ДО (если потребуется единое окно для администратора).

## Надёжность и эксплуатация

- Метрики/счётчики по регистру очереди для внешнего мониторинга (по согласованию с инфраструктурой).
- Дополнительные проверки прав на форму очереди и регистр при типовых ролях ДО.

## Интеграции

- Поддержка дополнительных облачных провайдеров при сохранении единого контракта «текст в промпт» для PDF.

## Документация и сопровождение

- Видео или скриншоты к `docs/INSTRUCTIONS.md` для типовых сценариев (по запросу).
