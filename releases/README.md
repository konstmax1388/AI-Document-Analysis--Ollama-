# Релизы расширения «ИИ анализ документов (Ollama)»

В каталоге публикуется готовый файл расширения **`.cfe`** для загрузки через **Конфигуратор → Расширения → Загрузить из файла**.

| Файл | Назначение |
|------|------------|
| `ИИАнализДокументовOllama.cfe` | Собранное расширение (бинарный артефакт) |

**Версия в исходниках** задаётся в `DOCFSKDEVII.ИИАнализДокументовOllama/src/Configuration/Configuration.mdo` (элемент `version`). После изменения метаданных пересоберите `.cfe` из EDT и обновите файл в этом каталоге.

Файлы `*.log` в `releases/` не коммитятся (см. `.gitignore`) — это локальные логи сборки.
