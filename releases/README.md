# Релизы расширения «ИИ анализ документов (Ollama)»

В каталоге публикуется готовый файл расширения **`.cfe`** для загрузки через **Конфигуратор → Расширения → Загрузить из файла**.

| Файл | Версия в метаданных | Назначение |
|------|---------------------|------------|
| `ИИАнализДокументовOllama2.0.cfe` | **0.5.5.0** | Актуальная поставка (облако + Ollama, настройки интеграции). См. [`docs/RELEASE_II_OLLAMA_2.0.md`](../docs/RELEASE_II_OLLAMA_2.0.md) |
| `ИИАнализДокументовOllama.cfe` | *(устарело)* | Прежнее имя файла; при отсутствии в каталоге используйте строку выше |

**Версия в исходниках** задаётся в `DOCFSKDEVII2.ИИАнализДокументовOllama/src/Configuration/Configuration.mdo` (элемент `version`). После изменения метаданных пересоберите `.cfe` из EDT и обновите файл в этом каталоге.

Файлы `*.log` в `releases/` не коммитятся (см. `.gitignore`) — это локальные логи сборки.
