# ИИ анализ документов (Ollama) — расширение для 1С:Документооборот

В репозитории публикуются только **исходники расширения** (проект EDT) и **актуальный релизный файл `.cfe`**.

| Что | Где |
|-----|-----|
| Исходники EDT | `DOCFSKDEVII2.ИИАнализДокументовOllama` |
| Сборка для установки | `releases/ИИАнализДокументовOllama2.0.cfe` |

Номер версии смотрите в метаданных расширения (`Configuration.mdo` / свойство версии в конфигураторе).
